Shadow Mapping & Multi-Light Rendering

Project Overview

This project demonstrates Shadow Mapping using OpenGL 3.3 and C++. It features a custom implementation of shadow generation for multiple light sources.

  • Multiple Lights: Additive blending of two distinct light sources (Green and Blue).
  • Shadow Mapping: Uses custom Framebuffers and Depth Textures.
  • PCF Filtering: Softens shadow edges for realism.
  • Directional Lighting: Corrected projection to simulate sun-like parallel rays.
